Detailed Description

This is the source file for the functions that have to do with TREE_NODEs.

It contains a function that creates a TREE_NODE, a function to create a child of a TREE_NODE , a function that calculates the manhattan value and a method to print the moves that created the node. Also contains two helping functions that are used be the makeChild function.
